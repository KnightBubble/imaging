imaging
=======

Drawing image in your terminal whithout node-canvas

## Installing

For [node](http://nodejs.org) with [npm](http://npmjs.org):
```bash
npm install imaging -g
```

## Usage

```bash
imaging https://www.google.com.hk/images/srpr/logo11w.png
```
